Simple Audio Player

Desktop audio mixer application with dual-deck audio player built using JUCE Framework and C++17

About the Project

Simple Audio Player is a professional desktop audio mixer application featuring a dual-deck audio player interface. Built using the JUCE Framework and C++17, this project demonstrates advanced audio processing capabilities, modern C++ practices, and user interface design.

Features

  • Dual-deck audio player interface
  • Play, Pause, Stop, and Loop controls
  • Playback speed control
  • Volume and gain control
  • Metadata extraction from audio files
  • Keyboard shortcuts for quick access
  • Professional audio mixing capabilities

Technologies

  • C++17: Modern C++ features and best practices
  • JUCE Framework: Cross-platform C++ application framework for audio applications
  • Audio Processing: Real-time audio manipulation and playback
  • Object-Oriented Design: Clean architecture and modular code structure

Project Highlights

This project showcases my ability to work with complex audio processing libraries, implement real-time audio manipulation, and create intuitive user interfaces. The dual-deck design allows for professional audio mixing workflows, while the comprehensive control set provides flexibility for various use cases.
